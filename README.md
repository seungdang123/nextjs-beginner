# NextJS Introduction

• Library vs Framework
Library : 개발자가 자기 입맛에 맞게 라이브러리를 가져와서 사용한다. 파일 구조 등을 원하는대로 마음대로 설정하면서 개발가능하다.
Framework: 개발자는 프레임워크가 제안하는 규칙에 맞게 코드를 짜야한다. 파일 구조 등을 마음대로 바꾸는 등의 프레임워크의 심기를 건드리는 행동을 한다면 에러가 발생한다.
